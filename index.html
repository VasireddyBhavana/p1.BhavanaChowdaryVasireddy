Project 1 REPORT

Control Interface

The microwave has all the basic options like cook, defrost, timer, start/stop etc available for the user so that they can use the microwave as per their need.

MicroWave Functioning

Microwave has 2 modes of operation where the user can select based on his requirement either from the instant cooking options or by selecting the timer manually.



Common Issues with the current design


Multiple Unused Options

It is recommended to eliminate uncommonly used options for same operation like multiple Timer options, multiple defrost options. It is also recommended to get all the those otpions within a single button. This is the recommendation that I would make to eliminate multiple unused buttons.


Power Level Button

Power level button is rarely used. Because we set the required timer based on the type of food.It can be used only in situations where there are frequent fluctuations which is very rare/


Time cook Button

Time cook is not essential as we already have an option to set the time in the form of the button +Add 30 sec.


Common Uses:

  1. Reheating food
  2. Boiling water
  3. Cooking
  4. Baking
  5. Disinfect kitchen items
  6. Heating beauty products

Rare Uses:

  1. Time Defrost
  2. Weight Defrost
  3. Kitchen Timer
  4. Power
  5. Popcorn

Sequence of actions to be performed:

  1. User opens the door
  2. place the food on the turntable
  3. Select from the options based on his need
  4. Set the Timer
  5. Wait for the Timer to complete
  6. After you get Notified by a sound remove the food

FeedBack

The user can obtain the feedback from the interface once the operation selected is completed. In general, the feedback is obtained in the form of a "Beep" sound confirming that the selected action is completed


Common Mistakes:

  1. Too many unnecessary keypad buttons, The user gets confused with multiple options for the same operatin
  2. Frequent heat sparkling in microwave
  3. Fuses keep blowing in a microwave
  4. Microwave and kitchen smells foul after cooking
